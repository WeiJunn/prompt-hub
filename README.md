# Prompt 分类仓库

这个仓库用来整理和维护一批可直接复制使用的中文 prompt，并按领域分类存放。

## 当前分类

### `求职/`
求职相关 prompt，包括：
- `岗位洞察prompt.md`
- `简历优化prompt.md`
- `面试准备prompt.md`

## 使用方式

1. 进入对应分类文件夹
2. 选择对应 prompt 文件
3. 复制完整 prompt 到 AI 工具中
4. 填入 JD、简历或补充背景
5. 根据输出继续迭代
